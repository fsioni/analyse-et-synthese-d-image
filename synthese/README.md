# TP noté - Synthèse d'images 2022/2023

## Dépendances

- Python 3.9+
- OpenCV 4.7+
- CMake 3.21+

## How to run
- In CLion : 
    - To run the program :
      - Load CMakeLists.txt
      - Click on run to launch `synthese`
    - To run test :
      - Add a configuration to run `Google_Tests_run`


- In terminal :
    - `mkdir build && cd build`
    - `cmake ..`
    - `make`
    - `./synthese`
